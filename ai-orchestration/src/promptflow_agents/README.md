# XRS NEXUS AI Orchestration Agents (Prompt Flow)

This directory contains modular Python Prompt Flow agents for metadata-driven, AI-orchestrated integration.

## Agents
- Metadata Intelligence Agent
- Logical Flow Generator Agent
- ETL Strategy Agent
- Runtime Telemetry Agent
- Lineage & Impact Analysis Agent

Each agent:
- Is implemented as a Python Prompt Flow module
- Has a prompt template, input/output JSON contract, and error handling
- Logs with correlation IDs for observability
- Is designed for production, modularity, and explainability
