# Synthetic Enterprise Metadata Dataset

This folder contains realistic synthetic metadata for enterprise systems, used for development, testing, and demos of XRS NEXUS.

## Systems Covered
- SAP (tables, CDC, IDocs)
- Salesforce (objects, incremental loads)
- REST APIs (OpenAPI-based)

## Features
- Schemas
- CDC flags
- SLAs
- Ownership
- Dependencies
- Data quality rules

## Usage
- Ingest these JSON files into Microsoft Fabric Lakehouse as Delta tables for platform-wide use.
- Use as test input for metadata ingestion, AI agents, pipeline generation, and lineage modules.
